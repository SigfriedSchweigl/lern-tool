Aufgabe:
Schreibe eine Methode findMax(int[] numbers), die das größte Element in einem Array zurückgibt.
Denkhinweis:
Überlege, mit welchem Wert du den Vergleich beginnen kannst.
Strategie:
Speichere zuerst ein vorläufiges Maximum und gehe dann mit einer Schleife durch die restlichen Elemente.
int max = numbers[0];
for (int i = 1; i < numbers.length; i++) { if (numbers[i] > max) { max = numbers[i]; } }
public static int findMax(int[] numbers) { int max = numbers[0]; for (int i = 1; i < numbers.length; i++) { if (numbers[i] > max) { max = numbers[i]; } } return max; }
